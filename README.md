## LinkedIn

### 1. Thought Leadership Post
```
Write a LinkedIn post about [topic] that positions me as an expert.
My role: [your title/niche]
Key insight: [your unique perspective]

Rules: Start with a bold statement or question. Use short paragraphs (1-2 sentences).
Include a personal story or example. End with a question to drive comments.
1200-1500 characters max.
```

### 2. Storytelling Post
```
Turn this experience into a compelling LinkedIn story:
What happened: [event/situation]
The lesson: [what I learned]
How it applies to [my audience/industry]

Format: Hook > Setup > Conflict > Resolution > Lesson > Call to action.
Use line breaks between paragraphs. No hashtags in the body.
```

### 3. Carousel Post Script
```
Create a LinkedIn carousel (document post) on [topic]:
Slide 1: Hook/title that stops the scroll
Slides 2-8: One key point per slide (keep text large and minimal)
Slide 9: Summary/recap
Slide 10: CTA (follow me for more)

For each slide: headline + 1-2 supporting sentences.
```

### 4. Comment Conversation Starters
```
I just posted about [topic] on LinkedIn. Generate 5 thoughtful comments I could leave
on OTHER people's posts about [related topic] to drive traffic back to my profile.

Rules: Add genuine value. Don't be promotional. Share a unique perspective or ask a
smart question. 2-3 sentences each.
```

## X / Twitter

### 5. Hook Tweet
```
Write a tweet about [topic] that stops the scroll.
My audience: [who follows me]
Goal: [engagement/authority/humor]

Rules: Under 280 characters. Start with a bold claim, surprising stat, or provocative
question. No hashtags in the body. One idea per tweet.
```

### 6. Educational Thread
```
Write a 7-tweet thread teaching [topic]:
My expertise: [what I know]
Target audience: [who benefits]

Format:
Tweet 1: Hook (why this matters, what they'll learn)
Tweets 2-6: One lesson per tweet (numbered, with examples)
Tweet 7: Summary + CTA (follow for more, bookmark this)
```

### 7. One-Liner Wisdom
```
Write 10 tweetable one-liners about [topic/industry]:
- Mix of advice, observations, and humor
- Each under 180 characters
- Easy to screenshot and share
- Sound like wisdom from experience, not AI
```

### 8. Quote Tweet Template
```
I want to quote-tweet [paste original tweet]. Write 5 different quote tweet options:
1. Add a unique insight
2. Agree and amplify with personal experience
3. Respectfully challenge one point
4. Add context that makes the original more valuable
5. Funny/witty take

Each under 280 characters. Sound human, not corporate.
```

## Instagram & TikTok

### 9. Reel Script (60 seconds)
```
Write a script for a 60-second [Instagram Reel / TikTok] about [topic].
My niche: [your niche]
Style: [educational / entertaining / inspirational]

Format:
- Hook (first 3 seconds): [attention grabber]
- Problem: [what your audience struggles with]
- Solution: [your tip/advice in 3 steps]
- CTA: [what to do next]

Include text overlay suggestions for each section.
```

### 10. Carousel Post (Instagram)
```
Create a 10-slide Instagram carousel about [topic]:
Slide 1: Bold title that makes people swipe
Slides 2-9: One tip per slide with a short explanation
Slide 10: "Save this for later" + CTA

For each slide: suggest headline text and body text (keep minimal — Instagram is visual).
Include relevant emoji but don't overdo it.
```

### 11. Caption with Hook
```
Write an Instagram caption for a post about [topic]:
My brand voice: [describe your tone]
Goal: [likes/saves/shares/comments]

Structure:
- Hook (first line — this shows before "more"): [make it irresistible]
- Value: 3-5 key points or a short story
- CTA: Ask a question or tell them to save/share
- Hashtags: 10-15 relevant ones (mix of small, medium, large)

Keep under 2200 characters.
```

### 12. TikTok Duet/Stitch Idea
```
I want to stitch/duet a video about [topic from another creator].
My angle: [what I'd add — agreement, pushback, additional tips]
My niche: [your niche]

Write a script that: hooks in the first 2 seconds, transitions smoothly from the
original, adds genuine value (not just "I agree"), and ends with my take or a
question for viewers. Under 45 seconds.
```

## YouTube

### 13. Video Script Outline
```
Create a YouTube video outline for: "[video title]"
Target length: [5/10/15] minutes
Audience: [who watches]

Include:
- Hook (first 30 seconds — why they should keep watching)
- Intro (brief, establish credibility)
- Main content (3-5 key sections with timestamps)
- B-roll suggestions for each section
- CTA placement (subscribe, like, comment)
- End screen suggestion (next video to watch)
```

### 14. Thumbnail + Title Combos
```
I'm making a YouTube video about [topic].
My channel niche: [your niche]

Generate 5 thumbnail + title combinations that maximize CTR:
- Title: under 60 characters, includes curiosity gap or number
- Thumbnail: describe the visual (what text, expression, colors)
- Why it works: explain the psychology behind each combo

Study these successful channels for style reference: [list 2-3 channels in your niche]
```

### 15. YouTube Shorts Script
```
Write a YouTube Shorts script (under 60 seconds) about [topic]:
Hook: [first 3 seconds — pattern interrupt]
Content: [one clear tip or insight]
Payoff: [satisfying conclusion]
CTA: [subscribe/watch full video]

Rules: Fast-paced. No fluff. Every sentence must earn its spot.
Keep text overlays to 3-5 words max.
```

## Cross-Platform Strategy

### 16. Content Repurposing
```
I wrote this [blog post / LinkedIn post / thread]: [paste content]

Repurpose it into:
1. A tweet thread (5-7 tweets)
2. An Instagram carousel (8-10 slides)
3. A TikTok/Reel script (60 seconds)
4. A LinkedIn post (different angle than original)
5. An email newsletter snippet (3 paragraphs)

Adapt the format and tone for each platform while keeping the core message.
```

### 17. 30-Day Content Calendar
```
Create a 30-day social media content calendar for [your niche]:
Platforms: [list platforms]
Posting frequency: [how often per platform]
Content pillars: [3-4 themes you want to cover]

For each day: platform, content type (post/reel/story/thread), topic, and hook.
Mix educational, entertaining, and promotional content (80/10/10 rule).
Include engagement activities (commenting on others' posts, responding to DMs).
```

### 18. Bio Optimizer
```
Rewrite my social media bio for [platform]:
Current bio: [paste it]
My niche: [what you do]
Target audience: [who you want to attract]
Goal: [followers/clients/community]

Requirements: Clear value proposition in first line. Include social proof if possible.
Has a CTA. Fits platform character limits. Uses relevant keywords for search.

Give me 3 versions: professional, personality-driven, and minimalist.
```

### 19. Viral Hook Generator
```
Generate 20 hook ideas for [your niche] content:
- 5 "Most people" hooks ("Most people waste 3 hours a day on...")
- 5 "Unpopular opinion" hooks
- 5 "I spent [time] doing [X] so you don't have to" hooks
- 5 "The [X] hack nobody talks about" hooks

Each should work as the first line of a post, reel, or thread.
Make them specific to [your niche], not generic.
```

### 20. Analytics Interpreter
```
Here are my social media analytics for the past [time period]:
[paste key metrics — impressions, engagement rate, follower growth, top posts]

Analyze:
1. What content type performed best and why?
2. What day/time drove the most engagement?
3. Which posts should I make more of?
4. What should I stop doing?
5. Give me 3 specific experiments to run next month based on this data.
```
